# Cooperative Management System

A full-stack MERN Cooperative Management System built with React, Node.js, Express, and MongoDB.

## Features

- Members Management
- Contributions Management
- Loan Management
- Reports Dashboard
- CRUD Operations
- REST API Integration
- Responsive Dashboard UI
- Sidebar Navigation
- Real-time Updates

## Technologies Used

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Project Structure

```bash
frontend/
backend/
```

## Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

## API Endpoints

### Members
- GET /api/members
- POST /api/members
- PUT /api/members/:id
- DELETE /api/members/:id

### Contributions
- GET /api/contributions
- POST /api/contributions

### Loans
- GET /api/loans
- POST /api/loans
